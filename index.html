GDScript_to_GDScript2.0
Welcome to the GDScript to GDScript 2.0 conversion tool! This program assists in upgrading your GDScript code to the latest version, introducing new features and improvements.

To get started:

  1. Input your GDScript code into the tool.
  2. Follow the prompts to apply suggested changes and optimizations.
  3. Review the converted code and make any necessary adjustments.
  4. Enjoy the enhanced features and performance of GDScript 2.0!

    5. [please notice that the translation is not always to 100% correct]

About This project

The project aims to improve the reusability of code from Godot 3 in Godot 4, thereby making it easier to translate by hand. With the present name changes from Godot 3 to Godot4, scripts can be easily translated and developers can save time and effort, resulting in faster development and fewer errors.
